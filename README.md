# bom-warnings widget for Tesserae

Australian Bureau of Meteorology weather warnings (severe storms, fire weather, marine, etc.) by state. Pulls the BoM warnings feed and shows current warnings with type, severity, and headline. No API key.

Set the cell's `state` (NSW, VIC, QLD, SA, WA, TAS, ACT, NT) and the widget shows live warnings; cancelled warnings are hidden by default.

Install via Settings → Widgets → Browse community widgets on [Tesserae](https://github.com/dmellok/tesserae).

## Folders shipped

- `sky_bom_warnings`

Previously bundled in [`tesserae-sky`](https://github.com/dmellok/tesserae-sky); carved into its own repo for v0.1.1+ so the bundle stays focused on the celestial / aerial widgets and Aussie users can install BoM warnings without the rest.
